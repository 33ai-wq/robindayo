Kalau API ini berguna untukmu — balik lagi. Kalau tidak, semoga harimu menyenangkan.

Filosofi

Kenapa halaman ini cuma HTML polos

Tanpa front-end

Tidak ada Cloudflare Pages, tidak ada React, tidak ada bundler. GitHub Pages sudah lebih dari cukup untuk etalase. Biaya operasional mendekati nol.

Tamu = transaksi

Setiap request ke endpoint adalah micro-payment otomatis lewat protokol x402. Kalau kamu berhenti datang, kami berhenti dihitung. Tidak ada auto-billing, tidak ada trial tersembunyi.

Harga jujur

Harga USDC sama dengan harga di x402scan. Tidak ada markup RobinDayo. Kalau kamu menemukan markup, itu bug — dan akan dikembalikan.

Sumber terbuka

Seluruh logika worker tersedia publik. Repo ini adalah etalase; worker sumber ada di repositori terpisah. Silakan audit.

Pembukuan

Kas rumah (treasury)

Setiap pembayaran yang masuk ke worker dicatat di kas Robinhood Chain di bawah. Saldo di bawah dibaca langsung dari RPC mainnet; kalau angkanya nol, ya memang nol.
